***Project Setup Guide***

**Prerequisites**
- Download [XAMPP](https://www.apachefriends.org/download.html) from the official website.
- Install XAMPP on your system.

**Installation Steps**
1. Download the project zip file.
2. Unzip the downloaded file.
3. Copy the unzipped project folder.
4. Navigate to C:\xampp\htdocs\ in your system.
5. Paste the copied project folder into the htdocs directory.

**Running the Project**
1. Open the XAMPP Control Panel by searching for "XAMPP" in the search bar and running it.
2. Start the Apache and MySQL modules by clicking on the "Start" button next to each.
3. Once Apache and MySQL are running, open a web browser.
4. In the web browser's address bar, type localhost/<folder_name> where <folder_name> is the name of your project folder.
   Example: If your project folder is named "project", type localhost/project.
5. Hit Enter to run your project locally.

**Additional Notes**
- Make sure XAMPP's Apache and MySQL services are running whenever you want to access your project.
- Ensure that your project files are placed inside the htdocs directory for proper execution.
